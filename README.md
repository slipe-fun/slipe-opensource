# Slipe app repo

Official repository of Slipe app contains code, assets and etc.

## Folders

- Components
- Constants
- Hooks
- Lib
- Patches
- Public
- Src
- Src-tauri

## Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/slipe-fun/app.git
   ```

2. Navigate to the landing directory:

   ```bash
   cd app
   ```

3. Install dependencies:

   ```bash
   npm i
   ```

4. Run project:

   ```bash
   npx tauri dev
   ```
